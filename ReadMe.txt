Este repositório contém as atividades desenvolvidas durante o semestre de 2023-1 na disciplina de Programação de Computadores 2 / Programação Orientada a Objetos, ministrada pela professora Luciana.

Sobre o Curso
Neste curso, exploramos os princípios da programação orientada a objetos (POO) e suas aplicações no desenvolvimento de software. Durante o semestre, adquirimos conhecimentos e habilidades essenciais para projetar, implementar e manter sistemas de software eficientes e modulares.

Atividades
Alguns topicos de atividades realizadas ao longo do curso:

1. Introdução à POO
Exploramos os conceitos fundamentais da programação orientada a objetos com um programa de Baralho, incluindo classes, objetos= e encapsulamento. Implementamos exemplos básicos para entender como os princípios de POO se aplicam na prática.

2. Projeto de Classes
Desenvolvemos um projeto de classe onde aplicamos os conceitos aprendidos. Criamos classes para modelar um sistema de gerenciamento de contatos.

3. Herança e Polimorfismo
Exploramos a relação de herança entre classes e a capacidade de polimorfismo em Java. Implementamos um sistema de Espaço2D para demonstrar como a herança e o polimorfismo podem simplificar o desenvolvimento.


Como Usar
Clone este repositório para sua máquina local usando o comando: git clone https://github.com/gabaraujodev/AtividadesPc2.git

Navegue para a pasta do projeto: cd AtividadesPc2

Consulte os diretórios das atividades para encontrar os códigos e recursos correspondentes.

Para executar um código, siga as instruções contidas no diretório da atividade específica.
